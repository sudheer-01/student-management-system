Faculty Responsibilities & Instructions

  1. Enter Marks: Enter student marks for the selected exam accurately. Marks must be within the permitted range for that exam.
  2. View Marks: Review marks entered for individual exams to ensure correctness.
  3. View Overall Marks: Analyze student performance across all exams. You can add calculated columns, apply filters, export data, or print reports.
  4. Edit Marks (HOD Approval Required): If a correction is required, update the marks and provide a clear reason. The request will be sent to the HOD for approval.
  5. Maintain Data Integrity: Enter and update marks responsibly. All changes are monitored and recorded.
